## 快速入门

### 系统简介

AI销售助手是一款智能化的销售管理工具，帮助您：
- 高效管理客户信息
- 追踪销售机会进展
- 快速生成专业报价
- 记录跟进过程
- AI辅助销售决策

### 首次使用

1. 联系管理员获取账号
2. 使用账号登录系统
3. 完善个人资料
4. 开始添加客户信息

## 登录注册

### 登录

1. 打开系统网址
2. 输入用户名和密码
3. 点击"登录"按钮
4. 成功后进入仪表盘

![登录界面示意]

### 注册新账号

1. 在登录页面点击"注册"
2. 填写注册信息：
   - 用户名（3-50字符）
   - 邮箱地址
   - 密码（至少6字符）
   - 姓名
3. 点击"注册"完成

### 修改密码

1. 点击右上角用户头像
2. 选择"个人设置"
3. 点击"修改密码"
4. 输入旧密码和新密码
5. 点击"确认修改"

## 客户管理

### 客户列表

**功能说明**：查看和管理所有客户信息

**操作步骤**：
1. 点击左侧菜单"客户管理"
2. 查看客户列表

**筛选功能**：
- 搜索框：输入关键词搜索客户名称、联系人、公司
- 行业筛选：按行业分类筛选
- 来源筛选：按客户来源筛选

### 添加客户

1. 点击"新建客户"按钮
2. 填写客户信息：

| 字段 | 说明 | 必填 |
|------|------|:----:|
| 客户名称 | 客户公司或项目名称 | ✓ |
| 联系人 | 主要对接人姓名 | |
| 联系电话 | 联系方式 | |
| 邮箱 | 电子邮箱 | |
| 公司名称 | 所属公司 | |
| 行业 | 行业分类 | |
| 来源 | 客户来源渠道 | |
| 地址 | 公司地址 | |
| 备注 | 其他说明 | |

3. 点击"保存"

### 编辑客户

1. 在客户列表中找到目标客户
2. 点击"编辑"按钮
3. 修改需要更新的信息
4. 点击"保存"

### 删除客户

1. 在客户列表中找到目标客户
2. 点击"删除"按钮
3. 确认删除操作

> ⚠️ **注意**：删除客户会同时删除相关的销售机会和跟进记录，请谨慎操作。

### 查看客户详情

1. 点击客户名称进入详情页
2. 查看客户基本信息
3. 查看相关销售机会
4. 查看跟进记录
5. 查看报价历史

## 销售机会

### 机会列表

**功能说明**：管理所有销售机会，追踪销售进展

**机会阶段**：
- 🟢 初步接触 - 客户初步了解（概率20%）
- 🔵 需求确认 - 明确客户需求（概率40%）
- 🟡 方案报价 - 已提供方案（概率60%）
- 🟠 商务谈判 - 进行商务洽谈（概率80%）
- ✅ 成交 - 成功签单（概率100%）
- ❌ 失败 - 未能成交（概率0%）

### 创建机会

1. 点击"新建机会"
2. 选择关联客户
3. 填写机会信息：

| 字段 | 说明 |
|------|------|
| 机会名称 | 项目名称 |
| 金额 | 预计成交金额 |
| 阶段 | 当前销售阶段 |
| 预计成交日期 | 预计成交时间 |
| 描述 | 项目详细说明 |

3. 点击"保存"

### 更新机会阶段

**方法一：拖拽方式**
- 在看板视图中，将机会卡片拖拽到对应阶段列

**方法二：编辑方式**
1. 点击机会卡片
2. 点击"编辑"
3. 选择新阶段
4. 保存

### 销售漏斗

**功能说明**：可视化展示各阶段机会分布

**查看步骤**：
1. 进入"销售机会"
2. 点击"漏斗分析"
3. 查看各阶段统计：
   - 机会数量
   - 金额分布
   - 转化率

## 报价管理

### 创建报价单

1. 点击"新建报价"
2. 选择客户
3. 添加报价项目：
   - 项目名称
   - 描述
   - 数量
   - 单价
4. 系统自动计算总价
5. 设置报价有效期
6. 添加备注（可选）
7. 点击"保存"

### 报价状态

| 状态 | 说明 |
|------|------|
| 草稿 | 正在编辑 |
| 已发送 | 已发送给客户 |
| 已接受 | 客户已接受 |
| 已拒绝 | 客户已拒绝 |
| 已过期 | 超过有效期 |

### 发送报价

1. 打开报价单详情
2. 点击"发送报价"
3. 确认发送

### 生成PDF

1. 打开报价单详情
2. 点击"导出PDF"
3. 下载PDF文件

---

## 跟进记录

### 创建跟进

1. 进入客户详情页
2. 点击"添加跟进"
3. 填写跟进内容：
   - 跟进方式（电话/邮件/面谈等）
   - 沟通内容
   - 下一步计划
   - 下次跟进时间
4. 保存

### 跟进提醒

系统会根据"下次跟进时间"自动提醒：
- 当天需要跟进的客户会在仪表盘显示
- 即将到期的跟进会提前提醒

### 跟进历史

在客户详情页可以查看：
- 所有跟进记录
- 跟进时间线
- 跟进统计

## 数据分析

### 仪表盘

**功能说明**：展示关键业务指标

**主要指标**：
- 客户总数
- 活跃机会数
- 总成交金额
- 转化率
- 本月新增客户
- 本月成交金额

### 销售报表

**访问路径**：数据分析 > 销售报表

**报表类型**：
1. **业绩趋势**
   - 按时间查看销售趋势
   - 对比不同时期业绩

2. **客户分析**
   - 客户来源分布
   - 行业分布
   - 地区分布

3. **机会分析**
   - 阶段分布
   - 成交率分析
   - 平均成交周期

### 导出报表

1. 选择报表类型
2. 设置时间范围
3. 点击"导出"
4. 选择导出格式（Excel/PDF）

## AI助手

### 功能介绍

AI助手可以帮您：
- 分析客户购买意向
- 生成销售建议
- 优化报价策略
- 预测成交概率

### 使用方法

**客户分析**：
1. 进入客户详情页
2. 点击"AI分析"
3. 查看分析报告：
   - 客户画像
   - 购买意向评分
   - 跟进建议

**报价建议**：
1. 创建报价单时
2. 点击"AI建议"
3. 获取智能报价建议：
   - 价格区间建议
   - 折扣策略
   - 成交概率

**对话助手**：
1. 点击右下角AI图标
2. 输入问题
3. 获取智能回答

**示例问题**：
- "这个客户的成交概率有多大？"
- "如何提高这个机会的转化率？"
- "建议的跟进策略是什么？"

## 常见问题

### 登录问题

**Q: 忘记密码怎么办？**

A: 
1. 在登录页点击"忘记密码"
2. 输入注册邮箱
3. 查收重置邮件
4. 设置新密码

**Q: 账号被锁定怎么办？**

A: 联系系统管理员解锁

### 数据问题

**Q: 如何批量导入客户？**

A:
1. 进入客户列表
2. 点击"导入"
3. 下载导入模板
4. 填写客户数据
5. 上传Excel文件

**Q: 数据可以导出吗？**

A: 可以。在各列表页面点击"导出"按钮，支持Excel格式。

### 功能问题

**Q: 如何查看历史数据？**

A: 使用筛选功能设置时间范围，或查看数据分析模块的历史报表。

**Q: 数据会自动保存吗？**

A: 编辑数据时需要手动点击保存。系统会自动记录所有操作历史。

### 权限问题

**Q: 为什么看不到某些功能？**

A: 不同角色有不同的权限，请联系管理员确认您的角色权限。

**Q: 如何申请更高权限？**

A: 联系组织管理员申请权限升级。

## 附录

### 数据字段说明

**客户信息**：
- 客户名称：必填，客户的唯一标识
- 联系人：主要对接人
- 联系电话：支持手机和座机
- 邮箱：用于发送报价和通知
- 行业：用于分类统计
- 来源：客户获取渠道

**销售机会**：
- 机会名称：项目或订单名称
- 金额：预计成交金额
- 阶段：销售流程阶段
- 概率：成交概率，影响加权金额
- 预计日期：预期成交时间

### 状态流转

**机会状态流转**：
```
初步接触 → 需求确认 → 方案报价 → 商务谈判 → 成交
                ↓
               失败
```

**报价状态流转**：
```
草稿 → 已发送 → 已接受
              ↓
            已拒绝
              ↓
            已过期
```
